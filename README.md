# 📊 Power BI Project

This repository contains real-world Power BI projects showcasing dashboard design, data modeling, and interactive analytics.  

### 🚀 Business Performance Dashboard  

An interactive Power BI dashboard built on a chocolate industry dataset to analyze company‑wide sales, shipments, and profitability

It includes:  
- 🎨 **Design & Layout**: Custom visuals, color theme, and recruiter-ready formatting  
- 🗄️ **Data Model**: Relationships between Sales, Products, Salesperson, and Date tables  
- 📊 **KPIs & Measures**: DAX formulas for metrics like Total Amount, Total Profit, Profit %, CY vs PY comparisons  
- 🔍 **Visuals & Charts**:  
   - Line charts for year-over-year trends (Amount & Boxes)  
   - Pie chart for geographic sales distribution  
   - Histogram for shipment distribution  
   - Bar chart for top 6 products  
   - Salesperson leaderboard table  
- 🎛️ **Interactivity**: Slicers for dynamic filtering (date, product, region), drilldowns for detail exploration  
- 🖼️ **Dynamic Insights**: Conditional formatting to highlight top performers and underperforming products
